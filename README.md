 ytsub
===================

# How to use

1. Install the CLI
    ```sh
    npm i -g git@github.com:victornpb/ytsub.git
    ```
2. Create a folder
    ```sh
    mkdir my_videos
    cd my_videos
    ```
3. Create a subscription.txt
    ```sh
    ytsub --create
    ```
4. Edit your subscriptions file
5. Start
    ```sh
    ytsub subscriptions.txt -t=86400
    ```
