 ytsub
===================

# How to use

1. Install the CLI
    ```sh
    npm i -g git@github.com:victornpb/ytsub.git
    ```
2. Create a folder for containing your files
    ```sh
    mkdir my_videos
    cd my_videos
    ```
3. Create a new server
    ```sh
    ytsub create
    ```
4. Edit subscriptions file
5. Start
    ```sh
    ytsub start subscriptions.txt -t=86400
    ```
